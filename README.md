## 网络系统结构
### 计算机网络的分类方式
按涵盖的地理范围划分：**局域网、城域网、广域网**
- 局域网(Local Area Network，LAN)
  >覆盖范围一般是方圆几千米之内，局部地区形成的一个区域网络
- 城域网(Metropolitan Area Network，MAN)
  >覆盖一个城市地理范围，是介于广域网与局域网之间的高速网络
  
  宽带城域网

- 广域网(Wide Area Network，WAN)
